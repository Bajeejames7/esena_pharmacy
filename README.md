# Esena Pharmacy - Responsive Glassmorphism UI

A modern pharmacy management system with glassmorphism design, featuring online ordering, appointment booking, and admin dashboard.

## Tech Stack

- **Frontend**: React, Tailwind CSS, Axios
- **Backend**: Node.js, Express, MySQL
- **Email**: Nodemailer
- **Auth**: JWT

## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env  # Configure your environment
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Features

- 📱 Fully responsive (mobile-first)
- 🎨 Glassmorphism UI design
- 💊 Product catalog (medicines, supplements, personal care)
- 📅 Appointment booking (dermatology, lab tests)
- 🛒 Online ordering with token-based tracking
- 📧 Email notifications
- 🔐 Admin dashboard with JWT auth
- 📊 Order & appointment management

## Database Setup

Run the SQL scripts in `backend/database/schema.sql` to create tables.
